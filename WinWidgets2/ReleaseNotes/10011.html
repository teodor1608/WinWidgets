WinWidgets Release Notes

Version 1.0.0.11

New:

  • Added Error System to better handle new and unknown issues
  • Added release date to updates
  • Added Systems Settings (You can change Screen Time System settings)
  • Added auto check for update feature to the Update System (checks every hour)
  • Added a cancel update button when the update is finished downloading and waiting for restart
  • Added update status tracking in between app sessions so you can now finish updates even after restarting or exiting the app
  • Added check for updates button
  • Added Notification System (You will now receive notifications about updates and more)

Changes:

  • Updated CefSharp to version 116.0.230
  • The version of the available update will now be displayed
  • Removed the placeholder expander from Widgets Settings
  • Changed the general update accent color
  • Changed the update cycle and release notes buttons
  • Removed menu items from the Settings menu that are not available yet
  • Removed menu items from the Analytics Report menu that are not available yet
  • Changed the spacing between items in the Analytics Dashboard to be relative

Fixes:

  • Fixed a Screen Time System issue causing the app to shut down when switching to a new day
  • Fixed a CEF issue causing the Settings window to load slower
  • Fixed an Update System issue which would cause updates to fail if the app has been closed during an update
  • Fixed spelling and capitalization in the update cycle texts
  • Fixed spelling in the Analytics Dashboard texts

Updater:

  • Increased the size of the Update Window for better readability
  • Fixed an issue with the position of the Release Notes
  • Improved the update clean up
