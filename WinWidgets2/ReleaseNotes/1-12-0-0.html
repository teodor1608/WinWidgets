WinWidgets Release Notes

Version 1.12.0.0

New:

  • Added the Skin Studio where you can customize the appearance of your widgets
  • Added the Notifications Hub which lets you view your notifications history from the lsat 48 hours
  • Added Notifications Settings to General Settings
  • Added Do Not Disturb which prevents notifications from being sent when in fullscreen mode
  • Added icons to the settings cards
  • Added Screen Time Widget which displays the current day's Screen Time System Data
  • Added Screen Time Widget settings to Widgets Settings
  • Added Info to some settings
  • Added tooltips to the Side Menu items
  • Added shadows to the Clock Widget
  • Added the Notifications Archive System
  • Added custom folder selection dialogs

Changes:

  • Updated CefSharp to version 117.2.40
  • Updated Microsoft.AppCenter to version 5.0.3
  • The DND toggle in the right click menu now works
  • Changed the tooltips on the About & Updates button and Login button
  • Changed the folder selection dialog when moving the WinWidgets Cache
  • Changed the way you disable the Screen Time System if the Screen Time Widget is enabled
  • Changed the versioning of the app (1/2.major.minor.fix)

Fixes:

  • Fixed a number of spelling errors within the Settings Window
  • Fixed an issue affecting the Update System (delivered in 1.0.11.1)
  • Fixed an issue with the Screen Time System which was preventing Sessions to be detected correctly resulting in an false sessions count
  • Fixed an issue with the widgets' reset function which was preventing the acrylic background form being applied
  • Fixed a number of spelling errors within the Settings Window
  • Fixed an issue when displaying the WinWidgets Cache location
  • Fixed an issue with the app icon tooltip which was preventing the tooltip from displaying anything

Updater:

  • None
