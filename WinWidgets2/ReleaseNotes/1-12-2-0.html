WinWidgets Release Notes

Version 1.12.2.0

New:

  • Added the Screen Time System Limits to the Screen Time Widget
  • Added the ability to expand the Screen Time Widget
  • Added keyboard activity detection to the Screen Time System
  • Added audio activity detection to the Screen Time System
  • Added Fullscreen activity setting to the Screen Time System Settings
  • Added Audio activity setting to the Screen Time System Settings

Changes:

  • Temporarily changed some UI elements in General Settings

Fixes:

  • Fixed an issue with the Screen Time System which was causing the inaccurate detection of Active Time
  • Fixed an issue with the Update System which was preventing users from receiving an update

Updater:

  • None
