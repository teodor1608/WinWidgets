WinWidgets Release Notes

Version 1.12.1.0

New:

  • Added the Screen Time System Blacklist
  • Added the Screen Time System Blacklist settings to the Screen Time System settings in Systems Settings
  • Added the Screen Time System Limits
  • Added the Screen Time System Limits settings to the Screen Time System settings in Systems Settings
  • Added a settings in General Settings to toggle the animated background in Settings
  • Added the ability for widgets to reset their location in case they are moved/left out of screen bounds
  • Added Lottie Animations

Changes:

  • Improved the Screen Time System apps filter

Fixes:

  • Fixed an issues with the tooltips font in a number of cases
  • Fixed an issues with the Screen Time Widget loading icons
  • Fixed an issue with the Internet Connection setting

Updater:

  • None
